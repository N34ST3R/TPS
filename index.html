Setup Guide

Get running in 5 minutes

1

Install TA-Lib

The C library for pattern detection. On Windows: pip install TA-Lib --find-links https://github.com/cgohlke/talib-build/releases. On Mac: brew install ta-lib.

2

Install Python deps

pip install -r requirements.txt — installs yfinance, TA-Lib wrapper, Quart, FastMCP, aiosqlite, httpx, and more.

3

Configure

cp .env.example .env then edit. Minimum: set ACCOUNT_SIZE=100. Optional: add Discord webhook, Solana wallets, Birdeye API key.

4

Run

python src/main.py — starts the dashboard on port 5000, MCP server on 8001, and background scanner.

5

Open Dashboard

Go to http://localhost:5000. Add tickers to your watchlist, click Scan, and watch signals appear.
